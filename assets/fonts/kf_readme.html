ＫＦひま字シリーズ 利用規約

last update 2018.04.01／KF STUDIO

オリジナル日本語フォント「ＫＦひま字」をはじめKF STUDIO制作のフォントは、MacintoshでもWindowsでも使えるOpenTypeフォントです。
KF STUDIOが制作したフォントは基本的に商用・非商用とも無料でお使いいただけます。
必ず利用規約、禁止事項をご確認ください。

  1. KF STUDIOが制作したフォントの著作権／所有権は、KF STUDIOに帰属します。
  2. 予告なくフォントの配布を中止したり、フォントおよび利用規約を改訂する場合があります。
  3. フォントのご使用によっていかなる損害が生じても、KF STUDIOは一切その責任を負いません。
  4. リンクしていただく際はKF STUDIOトップページにお願いいたします。http://www.kfstudio.net
  5. ＫＦひま字シリーズは （OTEdit for MAC （武蔵システム）で制作しました。

禁止事項

  1. フォントデータの改変・無断転載・再配布・販売・インターネット上へのアップロードは禁止します。
  2. フォントデータが埋め込まれた制作物の配布・販売を禁止します。（スマホアプリ・PDF制作物等。）
    • 1,2に関して、許可なくWebフォント化することも禁止です。
    • なおご利用を許可させていただいているリリース済の商品、アプリにつきましては、引き続きご利用いただけます。
  3. フォントを利用した文字主体の制作物の配布・販売を禁止します。
    • 文字主体にデザインされた印刷物素材、Web用素材
    • 文字主体にデザインされたTシャツなどのグッズ
    • 文字主体にデザインされたカッティングシート、ハンコ　　……など
  4. 商標登録・意匠登録等の権利が発生する場合は、使用できません。
  5. 「ＫＦひま字イラスト」を特定のキャラクターとしての利用は禁止します。
  6. 公序良俗に反するもの、違法なもの（著作権上の問題がある同人誌・動画等を含む）、アダルト関係等の印刷物・サイトでの使用は禁止します。
